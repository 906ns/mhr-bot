# ラズパイ移行ガイド（mhr-bot）

Windows で動かしている Bot を Raspberry Pi に移行して、**24時間自動起動で動かす**ための手順です。

## 必要なもの

- Raspberry Pi 本体（Pi 3 以降 / Pi Zero 2 W でもOK。この Bot はとても軽いです）
- microSD カード（8GB 以上）
- ネットワーク接続（Wi-Fi か有線LAN）

## 1. OS の準備（初回のみ）

1. Windows に [Raspberry Pi Imager](https://www.raspberrypi.com/software/) をインストール
2. OS は **Raspberry Pi OS Lite (64-bit)** を選択（画面不要のBot用途ならLiteで十分）
3. 書き込み前の歯車（詳細設定）で以下を設定しておくと楽です
   - ホスト名（例: `raspberrypi`）
   - **SSH を有効化**
   - ユーザー名とパスワード（以下の手順では `pi` として説明）
   - Wi-Fi の SSID とパスワード
4. SD カードに書き込み、ラズパイに挿して電源ON

## 2. Windows からラズパイに SSH 接続

PowerShell を開いて:

```powershell
ssh pi@raspberrypi.local
```

（つながらない場合はルーターの管理画面などでラズパイのIPアドレスを調べて `ssh pi@192.168.x.x`）

## 3. Bot のセットアップ

ラズパイ上で以下を実行:

```bash
# 必要なパッケージ
sudo apt update
sudo apt install -y git python3-venv

# リポジトリをクローン
git clone https://github.com/906ns/mhr-bot.git
cd mhr-bot

# 仮想環境を作成してパッケージをインストール
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
```

## 4. トークン（.env）の設定

Windows の `.env` の中身と同じものを作ります。ラズパイ上で:

```bash
echo 'DISCORD_TOKEN=ここにトークンを貼る' > .env
```

（または Windows の PowerShell から `scp .env pi@raspberrypi.local:~/mhr-bot/` でファイルごと転送してもOK）

## 5. 動作確認

```bash
.venv/bin/python bot.py
```

「ログイン完了: BotName#1234」と出れば成功。`Ctrl+C` で一旦止めます。

## 6. 自動起動の設定（systemd）

電源を入れるだけで Bot が起動し、落ちても自動で再起動するようにします。

```bash
sudo cp deploy/mhr-bot.service /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable --now mhr-bot
```

> ⚠️ ユーザー名が `pi` 以外の場合は、コピーした
> `/etc/systemd/system/mhr-bot.service` の `User=` と2箇所のパスを
> 自分のユーザー名に書き換えてから `daemon-reload` してください。

### 状態確認・ログ

```bash
systemctl status mhr-bot          # 動いているか確認
journalctl -u mhr-bot -f          # ログをリアルタイム表示（Ctrl+Cで抜ける）
```

### 停止・再起動

```bash
sudo systemctl stop mhr-bot
sudo systemctl restart mhr-bot
```

## 7. 移行完了後

- **Windows 側の Bot は止めてください**（同じトークンで2つ同時に動くと二重応答になります）
- Bot を更新したいときはラズパイ上で:

```bash
cd ~/mhr-bot
git pull
.venv/bin/pip install -r requirements.txt
sudo systemctl restart mhr-bot
```
